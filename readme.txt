Easy, fast, and simple listings management for realtors.

== Description ==

[Realtor Express](http://pulsarwebdesign.com/realtor-express) is a WordPress plugin designed specifically for realtors. By employing a simple and intuitive user interface, clean shortcodes, and a number of useful widgets the plugin provides maximum speed and agility when developing real-estate websites.

For more information, check out [pulsarwebdesign.com/realtor-express](http://pulsarwebdesign.com/realtor-express).

= Main Features =
* Simple and intuitive user interface.
* Listings management using WordPress internal post management system.
* Real time address validation using Google Maps API.
* Listings filter (search) widget.
* Mini-map widget.
* Interactive multi-map.
* Manage each listing's gallery using WordPress built in media manager.

== Installation ==

1. Upload the entire `realtor-express` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

For basic usage, check the plugin documentation page or have a look at the [plugin homepage](http://pulsarwebdesign.com/realtor-express). For a live demonstration visit [rex.pulsarwebdesign.com](http://rex.pulsarwebdesign.com).
